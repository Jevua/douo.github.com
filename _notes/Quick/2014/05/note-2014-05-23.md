### 加号前置

	+function() { console.log("Foo!"); }();

加号后面的部分将会被当成表达式（Expression）处理。除了加号外还可以加上去其他一元运算符（`-`, `!`, `~`），也能达到同样效果，等同于下面的表达式：

	(function() { console.log("Foo!"); })();
	// or
	(function() { console.log("Foo!"); }());

用加号还有个不同的作用，就是将返回结果转换为 `Number`

	+new Date()
